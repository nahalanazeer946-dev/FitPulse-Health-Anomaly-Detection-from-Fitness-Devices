
# FitPulse Health Anomaly Detection from Fitness Devices

## 📌 Project Overview
FitPulse is an AI-based health analytics system developed as part of an internship. It analyzes wearable fitness data such as heart rate, steps, sleep, and calories to detect anomalies and identify patterns.

## 📊 Features
- Anomaly detection in health data  
- User clustering based on activity  
- Time series forecasting  
- Interactive dashboard visualization  

## 🛠 Technologies Used
- Python  
- Streamlit  
- Pandas, NumPy  
- Machine Learning  

## 📁 Project Structure
- milestone1.py – Data preprocessing  
- milestone2.py – Modeling  
- milestone3.py – Anomaly detection  
- milestone4.py – Dashboard

streamlit run app.py
