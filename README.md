🏥 FitPulse Health Anomaly Detection from Fitness Devices

📌 Project Overview

FitPulse is an AI-based health analytics system developed as part of an internship.
It analyzes wearable fitness data such as heart rate, steps, sleep, and calories to detect anomalies and identify meaningful health patterns.

---

🚀 Features

- ✅ Data preprocessing of fitness datasets
- ✅ Feature extraction using TSFresh
- ✅ Clustering and pattern analysis
- ✅ Anomaly detection in health data
- ✅ Interactive dashboard visualization using Streamlit

---

🛠 Technologies Used

- Python
- Streamlit
- Pandas, NumPy
- Machine Learning
- Matplotlib / Plotly

---

🔄 Workflow

Upload CSV → Preprocessing → Feature Extraction → Clustering → Anomaly Detection → Dashboard Visualization

---

📂 Project Structure

- "preprocessing/milestone1.py" – Data preprocessing
- "pattern_extraction/" – Feature extraction and clustering
- "anomaly_detector/milestone3.py" – Anomaly detection
- "Insight_dashboard/milestone4.py" – Dashboard and report generation
- "main_app.py" – Integrated Streamlit application

---

📊 Dataset

The dataset consists of wearable fitness data including:

- Heart rate
- Steps
- Sleep
- Calories

---

▶️ How to Run

streamlit run main_app.py
